Call of Duty: Modern Warfare and Warzone get Verdansk dark mode, Zombie Royale for Halloween

On Oct. 20, Call of Duty: Warzone and Modern Warfare’s Halloween event officially begins. The Haunting of Verdansk will include new game modes, rewards, and skins from The Texas Chainsaw Massacre and Saw. The event runs from through Nov. 3. Perhaps the biggest inclusion in this Halloween event are the bundles themed after the Texas Chainsaw Massacre and Saw movie franchises, which include Leatherface and Billy the Puppet skins. Saw fans can also earn a special throwing knife called The Phlebotomozier, an assault rifle blueprint, a shotgun blueprint, a Cargo Truck skin, a tricycle charm, and “four other pieces of personalization content.” All of this is seemingly only available via the bundle.

PUBG Mobile new release: PUBG releases new maps & features in 1.1 Beta Update

PUBG Mobile version 1.1 is available for the beta testers and some new exclusive modes and PUBG new maps are added. Before any PUBG new update, the developers of PUBG Mobile release a beta version for players to test the new features and fix any bugs or glitches that might arise. A new game mode, a new item for the classic game mode, as well as a winter-themed event a brand new Metro Royale mode, winter festival, Black Market, and more will be introduced to the game with this update. There is also Customizable Loadouts and New Maps incoming.
